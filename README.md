# Переводчик фильма RU ↔ VI

Готовый PWA-пакет для iPhone.

## Список файлов

- `index.html` — основное приложение с 2 страницами:
  - страница 1: перевод, языки, крупная лента
  - страница 2: технические настройки
- `manifest.json` — настройки установки PWA
- `service-worker.js` — кэш и обновление приложения
- `icons/icon-192.png` — иконка 192×192
- `icons/icon-512.png` — иконка 512×512

## Что уже настроено

- Worker URL по умолчанию:

## Как выгрузить на GitHub

### Вариант через веб-интерфейс GitHub

1. Открой репозиторий.
2. Нажми `Add file` → `Upload files`.
3. Загрузи все файлы и папку `icons`.
4. Нажми `Commit changes`.

## Как подключить к Cloudflare Pages

1. В Cloudflare Pages выбери `Connect to Git`.
2. Выбери этот репозиторий.
3. Branch: `main`
4. Build command: оставить пустым
5. Output directory: оставить пустым или `/`
6. Нажми `Save and Deploy`

После этого схема такая:

`изменил файлы в GitHub → Commit changes → Pages обновился сам`

## Как обновлять PWA

Если меняешь важные файлы и хочешь принудительно обновить кэш, измени версию в `service-worker.js`:

```js
const CACHE_NAME = 'film-translator-v4';
```
