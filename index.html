Лента диалога

Что изменено в версии 3.6

1) Транскрибация и перевод разведены: сегмент появляется сразу, перевод обрабатывается в отдельной очереди.
2) Фраза теперь анализируется лингвистически: RU/VI определяется по буквам, диакритике и служебным словам, а не по активному модулю SpeechRecognition.
3) Активный модуль слушания остаётся только способом получить текст; источником языка считается сама фраза.
4) Нет поиска английского и других языков: только ru-RU и vi-VN.
5) v3.8: после готовой фразы следующий модуль выбирается по лингвистически определённому языку фразы: VI → слушать RU, RU → слушать VI.
5) История, настройки и адрес Worker сохраняются в localStorage.
6) Переключение VI→RU/RU→VI выполняется через безопасный stop→onend→start, без двойного старта.
7) Оставлен один HTML-файл для загрузки на Cloud Pages / GitHub Pages.
